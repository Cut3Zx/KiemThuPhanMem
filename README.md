# Bài tập Unit Test với JUnit 5 - Phân tích điểm học sinh

## 1. Chức năng
- `countExcellentStudents`: Đếm số lượng học sinh có điểm >= 8.0.
- `calculateValidAverage`: Tính điểm trung bình của các điểm hợp lệ (0-10).

## 2. Hướng dẫn chạy Test
1. Mở dự án bằng **Visual Studio Code**.
2. Đảm bảo đã cài đặt **Extension Pack for Java**.
3. Vào tab **Testing** (hình bình thí nghiệm) ở thanh công cụ bên trái.
4. Nhấn nút **Play** để chạy toàn bộ kiểm thử.

## 3. Kết quả
- Tất cả các ca kiểm thử (Trường hợp bình thường, Trường hợp biên, Trường hợp ngoại lệ) đều đã **Passed ✅**.
